# DevicesideAIModel

A sample device-side LLM chat app using CoreAI framework.



> [!important]
>
> This project uses Qwen3-4B for a demo, and **this project does NOT provide model files. Please download and place them yourself as described in [the instructions](./DevicesideAIModel/QwenAI/Readme.md).**

 You can choose other models to run.
